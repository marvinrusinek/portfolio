Portfolio

The following is a portfolio gallery of web development, UX design and programming projects that I have worked on professionally and as part of my studies. Each title below a project image is linked to its runnable project. When no project link is available upon clicking, you will be redirected back to the top. Most projects should have both “View Project” and “View Code” buttons displayed but please be aware that some projects may not have a project or code link available.

Clicking the “View Project” button will bring up a pop-up window with a larger image of the project, relevant project metadata, the problem encountered and its solution along with contextual information. Buttons are available near the bottom of the window to view the live project or to see the source code. When clicked, the “View Code” button on the portfolio section or in the pop-up window will bring you to the respective GitHub code repository where you can view the code and gain a sense of my process.
