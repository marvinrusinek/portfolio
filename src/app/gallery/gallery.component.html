Portfolio

This is my web developer portfolio. The portfolio gallery consists of my web development projects as well as UX design and programming projects that I have worked on professionally and as part of my studies. Most projects should have both “View Project” and “View Code” buttons below the project image, but please be aware that some projects may not have a project or code link available. Hovering over an image will bring up a button to click for more project information contained within a modal.

The modal consists of a screenshot of the project, project category, duration, technologies used, contextual information, the problem encountered and its solution. Clicking the “View Project” button will bring up the project in a new tab. When clicked, the “View Code” button on the portfolio section or in the modal will bring you to the respective GitHub code repository where you can view the code and gain a sense of my process. If any of these projects or skills are a match with your requirements, please feel free to contact me below.
